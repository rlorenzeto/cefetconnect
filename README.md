# Cefet Connect - Plataforma de Integração Acadêmica

O **Cefet Connect** é uma rede social exclusiva para o ecossistema do CEFET, projetada para centralizar informações, fomentar o networking e integrar alunos de diferentes cursos e períodos. O projeto nasce da necessidade de quebrar "bolhas sociais" e evitar que oportunidades acadêmicas se percam em chats informais.

## 🚀 O Projeto

A plataforma visa transformar a experiência universitária ao oferecer um ambiente digital onde a comunidade acadêmica pode interagir de forma organizada e eficiente.

### 🚩 Problemas que resolvemos
- **Dispersão de informação:** Evita que eventos e avisos importantes se percam no fluxo de mensagens de grupos de WhatsApp.
- **Isolamento social:** Conecta alunos que, embora frequentem o mesmo campus, muitas vezes não interagem por estarem em cursos ou períodos distintos.
- **Perda de histórico:** Garante que informações úteis permaneçam acessíveis mesmo após o fim dos semestres letivos.

## 🛠️ Tecnologias Utilizadas

O projeto utiliza uma stack moderna focada em escalabilidade e performance:

- **Front-end:** [React](https://reactjs.org/)
- **Back-end:** [Node.js](https://nodejs.org/) com [TypeScript](https://www.typescriptlang.org/)
- **Banco de Dados:** MySQL
- **Arquitetura:** Planejada para suportar módulos independentes e escaláveis.

## 📋 Módulos do MVP (Mínimo Produto Viável)

1. **Sistema de Usuários:** Cadastro, login e perfis personalizáveis.
2. **Feed de Publicações:** Espaço central para compartilhamento de novidades e interação (likes/comentários).
3. **Comunidades Acadêmicas:** Grupos segmentados por interesses, projetos ou cursos.
4. **Eventos:** Organização e divulgação de atividades promovidas pelas comunidades.
5. **Tela de Contatos:** Facilitador de networking e busca de membros da comunidade.
